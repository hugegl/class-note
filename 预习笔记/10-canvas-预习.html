一,准备工作 1,准备画布 HTML标签cancas即可创建画布; 1)设置画布的大小直接在标签内写width,height;不建议在css中设置,因为css设置的是像素点变大了,拉伸画布;不是整个画布变大了; 默认300px,150px;白色; 2,准备绘制工具 JS的api进行操作 script标签内写api; 1)jS中拿到画布标签; 2)获取上下文/绘制工具箱; var 工具箱 = 画布元素.getContext(参数); 参数:2d;2d的绘制方式;平面2d的绘图; web gl 绘制3d效果的技术; 3,利用工具绘图 1) 工具箱.moveTo(x,y);移动画笔至坐标x,y的地方; 2) 工具箱.lineTo(x,y);画出x至y的绘制路径/轨迹; 3) 工具箱.stroke();描边 二,常用api 1,线条默认状态 1)默认宽度 1px 2)默认颜色 黑色 肉眼看的状态时2px,灰色; 原因,对齐的点是线宽度的中心位置;会把线分成两个0.5px,显示的是会不饱和 解决方案:画笔前后移动0.5px; 2,指定画笔颜色 工具箱.strokeStyle = '颜色';//改过以后,默认就是这个状态了;如果重复设置多次,以最后一次的设置为准; 3,指定画笔宽度 工具箱.lineWidth = 数字 ;//改过以后,默认就是这个状态了;如果重复设置多次,以最后一次的设置为准; 4,开启新路径 工具箱.beginPath();//开启新的路径,前面的设置不生效,默认回到初始状态;可以解决样式覆盖的问题; 5,填充 工具箱.fill();//填充;默认黑色; 工具箱.fillStyle('颜色');设置填充的颜色; 缺点:起始点和lineTo的结束点,无法进行闭合缺角; 解决方法:自动闭合; 填充遵循的规则:非零环绕规则; 1)看一块区域是否填充; 2)从这个区域拉一条直线; 3)看和这条直线相交的轨迹; 4)如果是顺时针轨迹则+1,如果是逆时针轨迹则-1; 5)把值相加,如果值=0则不填充,如果值!=0则填充; 6,自动闭合: 工具箱.closePath();自动闭合路径;关闭路径; 7,画笔相关的api 1)线末端类型 工具箱.lineCap = '参数';butt默认没有;square 方形;round原型 2)相交的拐点 工具箱.lineJoin = '参数';miter默认尖角;round 圆角;bevel 斜面,平的; 3)设置虚线 工具箱.setLineDash(参数);[数字]虚和实使用同一个数字;[数字1,数字2,数字3] 实线数字1,虚线数字2,实线数字3然后重复1; 4)获取虚线宽度集合 工具箱.getLineDash();获取的是不重复的那一段的排列方式; 5)设置虚线偏移量 工具箱.lineDashOffset = '数字';//负值向右偏移;正值往左偏移 8,颜色填充 1)渐变,使用for循环,注意每次要画笔开启新路径; 9,获取当前画布的宽/高 工具箱.canvs.with/height;返回值就是当前这个画布的宽/高; 10,矩形相关//起点位置在矩形的左上角 绘制矩形的路径 工具箱.rect(x坐标,y坐标,宽,高);//不是独立路径,需要开启新路径; 绘制描边矩 工具箱.strokeRect(x坐标,y坐标,宽,高);//有自己的独立路径;默认是黑色; 填充描边矩 工具箱.fillRect(x坐标,y坐标,宽,高);//有自己的独立路径;默认是黑色; 清除矩形的内容 工具箱.clearRect(x坐标,y坐标,宽,高);//清除矩形内的内容,清除画布,将宽高设置成画布大小即可; 线性渐变方案 createLinearGradient 线性渐变;createRadialGradient 径向渐变 1)先创建渐变方法 var 渐变方法 = 工具箱.createLinearGradient(x坐标,y坐标,x结束的坐标,y结束的坐标) ;设定渐变的范围,开始坐标和结束坐标; 2)调用颜色渐变 渐变方法.addColorStop(数字,'颜色');//数字0是0%,1是100%的意思;渐变的范围原理和C3的线性渐变一样; eg: 渐变方法.addColorStop(0,'red'); 渐变方法.addColorStop(1,'blue');//从0开始是红色,1的地方是蓝色,中间是渐变; 三,圆 弧度:一种长度的描述;1个弧度=1个半径的长度; 一个圆有2*π个弧度的长度 一个角度有多少弧度? (2*π)/360 => π/180; 常用方法: 绘制路径 工具箱.arc(圆心x坐标,圆心y坐标,半径,起始弧度,结束弧度,true/false);方向默认是false顺时针方向;起始弧度在圆的x轴方向右边 四,文字 1,描边文字 工具箱.strokeText('文字',x坐标,y坐标);//起点在左下角; 2,字体样式和大小 工具箱.font = '尺寸 字体'; 3,填充文字 工具箱.fillText('文字',x坐标,y坐标,maxwidth);//起点在左下角;maxwidth设置文本最大的宽度,文本会自动收缩,一般不用; 4,对齐方式 工具箱.textAlgin = '参数';left/right/center/end/start默认;//基准对齐点是文字的起始坐标; 工具箱.direction = '参数';ltr/rtl(了解); 5,垂直对齐方式 工具箱.textBaseline = '参数';top/middle/bottom;//基于文字的起始坐标;默认是基线对齐,中文,英文,印度文的不一样; 6,获取文本的宽度 工具箱.measureText('文本').width;返回值是文本的宽度;工具箱.measureText('文本')返回值是一个对象; 五,图片 创建图片对象 var 变量名 = new Image();//创建一个image对象; 给图片对象增加属性 image对象.src = '路径';//需等图片加载完毕才能获取到,所以需要注册load事件;先注册加载事件,再设置图片路径; 1,绘制图片的三种方式 3个参数 工具箱.drawImage(图片对象,x坐标,y坐标) 5个参数 工具箱.drawImage(图片对象,x坐标,y坐标,图片宽,图片高);是缩放; 9个参数 工具箱.drawImage(图片对象,图片上的定位x坐标,图片上的定位y坐标,截取多少图片宽,截取多少图片高,画布x坐标,画布y坐标,图片缩放的宽,图片缩放的高);裁切; 六,转换//转换的都是坐标 1,移动 工具箱.translate(x坐标,y坐标);移动的是canvas的坐标; 2,缩放 工具箱.scale(x缩放倍数,y缩放倍数);缩放的的是canvas的坐标; 3,旋转 工具箱.rotate(弧度);旋转的是坐标; 案例知识点: Math.pow(基数,指数);//基数的指数幂; Math.sin(数字); Math.PI;//π
