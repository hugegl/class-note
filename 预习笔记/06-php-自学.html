一,服务器 相当于一个主机 二,web服务器软件 1,Nginx 反向代理 2,Apache php 3,IIS ASP.NET 4,Tomcat java 三,安装web服务器软件 severRoot 配置安装目录 四,服务端开发基础 1,单个网络情况 如果设备没有连接任何网络情况下,我们会有一个本地回环地址127.0.0.1 单个局域网下,网关给我们分配了一个地址,在这个范围内我们可以通过这个地址找到这个设备 五,域名 ip地址的别名,需要花钱注册 六,DNS服务器 通过域名访问到ip地址的过程叫DNS寻址(DNS解析) 七,hosts文件 操作系统在发起对DNS查询的时候,会优先检查本机本机中的hosts文件,如果这个文件中包含了对当前需要解析的域名的配置,则不会发起请求,直接使用; 但是本机修改的hosts文件只能影响本机 八,端口 每台计算机有0-65536个端口 每个软件启动就会开一个端口,我们把'占门'这个过程叫监听 Apache监听的是80端口 https监听的是443端口 配置文件中 Listen 80 就是监听80端口,可以监听多个端口 九,URL eg: https://zce.me:80/school/student?id=18&name=zce#photo https 协议 zce.me:80 host(主机) /school/student 请求路径 id=18&name=zce 请求的参数 #photo 锚点值 十,网站根目录 在配置文件中 DocumentRoot 网页根目录 / 代表所有文件夹,根目录
