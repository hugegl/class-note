// var max = arr[0]; //下面的 var i = 1;可提升比较性能 for ( var i = 0 ; i < arr.length ; i++) { //执行的代码; if( arr[i] > max ) { max = arr[i]; } } 7,当比较数组内部中元素的大小时,不能从外部引入一个元素, 故初始值为: 数组名[0] 即用数组中第一个元素做初始值,进行比较 8,倒叙遍历,添加到新数组中 eg: var arr = [1,6,3,26,3] var arr2 = []; for ( var i = arr.length-1; i >= 0 ; i--){ arr2.push(arr[i]); } 9,冒泡 老师的总结: 1)先来两个for循环 2)外层-1,里层-1 -j的原因: 外层-1 ; 确定外层比较的遍数的,5个值,确定4个就可以了, 最后一个默认是最小值,不需要比较,比较的次数正好是length-1 里层-1 ; 里面的元素进行两两比较,比较的次数正好是length-1 里层-j(外层遍历的下标变量);原因,里层比较一次会确认最大的值, 依次增加比较的次数,会依次确认倒数的不需要比较的元素的次数, 增加不需要比较的次数的频率和值刚好与j每次增加的次数吻合, 所以-j,优化性能.
